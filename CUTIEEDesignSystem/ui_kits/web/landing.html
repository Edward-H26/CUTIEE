The fix

Stop reasoning. Start replaying.

CUTIEE turns every completed task into a zero-cost procedural template, routes live decisions to the cheapest viable model, and prunes context to what still matters.

01

Learn once

The first run through a task pays full rate. Steps, selectors, and decisions are captured into a procedural template.

02

Replay forever

Every run after drops to $0 for any step the template covers. Only genuinely new decisions hit the model.

03

Route what's left

The remaining decisions route to the cheapest tier that can handle them — with every action logged and signed.
